## About
It's a small app in angular js, based on fountainJs webapp generator

### Installing

Run npm install

```
npm install
```

follow gulp conf file for usage

```
gulp build 
gulp test
gulp server
```
etc...

## Deployment

The app is deployed on [heroku ](https://niv-weather.herokuapp.com/)
